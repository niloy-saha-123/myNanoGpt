# My Experiment

A quick-test experiment using `comp560-nanoGPT` to train a baby character-level Shakespeare model.

## Config

- **Dataset**: `shakespeare_char`
- **Model**: 4 layers, 4 heads, 128 embedding size
- **Training**: 50 iterations (quick test), `max_iters=50`
- **Device**: Apple MPS
- **WandB Project**: `comp560-onboarding`
